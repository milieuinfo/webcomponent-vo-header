# vo-header

De standaard header voor websites en applicaties van de Vlaamse overheid.

## Installatie

```
npm install --save vo-header
```

## Gebruik
### Polymer 2
Maak gebruik van de component in de dist folder.

### Polymer 3
Maak gebruik van de component in de root folder.
